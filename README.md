# verse-portable-browser
A portable and minimal browser that has all you need for a safe and private session.

Not the best and not the worst browser - just better than MS Edge LOL

It has a built-in hardware acceleration, a New York fixed location and an auto fixed Youtube 4k quality - what's good I guess.

For more information on how to use it, read the readme.txt file.
